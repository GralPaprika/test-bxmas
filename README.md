# Test BX+

Photo gallery application built with modern Android best practices.

## Tech Stack

- **Kotlin** with Jetpack Compose
- **Clean Architecture** (Data → Domain → Presentation)
- **MVI Pattern** for state management
- **Hilt** for dependency injection
- **Retrofit** for API communication
- **Coil** for image loading
- **Material Design 3** theming

## Key Implementation Details

- **MVI Pattern**: State-driven UI with Intent-based user actions
- **Pagination**: Infinite scroll with efficient data loading
- **Error Handling**: Comprehensive error states with retry
- **Loading States**: Skeleton screens for better UX
- **Dependency Injection**: Hilt for clean, testable code and separation of concerns
